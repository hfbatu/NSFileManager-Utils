NSFileManager-Utils
===================

Useful class of NSFileManager

## Usage
```objective-c
[[NSFileManager defaultManager] document];
[[NSFileManager defaultManager] documentWithPath:@[@"sample", @"test.jpg"]];
    
[NSFileManager document];
[NSFileManager documentWithPath:@[@"sample", @"test.jpg"]];
```
